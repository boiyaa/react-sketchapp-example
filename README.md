# React Sketch.app example

[![Greenkeeper badge](https://badges.greenkeeper.io/boiyaa/react-sketchapp-example.svg)](https://greenkeeper.io/)

## How to use

Install the dependencies

```
npm install
```

Run with live reloading in Sketch

```
npm run render
```

Or, to install as a Sketch plugin:

```
npm run build
npm run link-plugin
```

Then, open Sketch and navigate to `Plugins â†’ react-sketchapp: Example`

### Run it in your browser

```
npm run web
```

Open a browser to `http://localhost:3000`

### Build it for web

```
npm run web:build
```
